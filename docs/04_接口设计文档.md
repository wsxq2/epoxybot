## 1. ROS2 接口设计

### 1.1 Topic接口

#### 1.1.1 机器人状态Topic

**Topic名称**: `/robot/state`

**消息类型**: `RobotState.msg`

```python
# RobotState.msg
std_msgs/Header header

# 基本状态
string robot_id              # 机器人ID: "sidewall", "floor", "epoxy"
string status                # 状态: "idle", "running", "error", "emergency_stop"
float32 battery_level        # 电池电量 0-100%
float32 progress             # 作业进度 0-100%

# 位置信息
geometry_msgs/Pose pose      # 当前位姿

# 作业信息
string current_task          # 当前任务: "inspect", "grind", "spray"
float32 spray_thickness      # 喷涂厚度 mm
float32 material_remaining   # 材料剩余 %

# 传感器状态
bool lidar_ok
bool camera_ok
bool imu_ok
bool plc_ok

# 故障信息
string[] errors              # 故障列表
```

**QoS配置**:
```yaml
reliability: RELIABLE
durability: TRANSIENT_LOCAL
history: KEEP_LAST
depth: 10
deadline: 100ms
```

---

#### 1.1.2 控制指令Topic

**Topic名称**: `/robot/cmd`

**消息类型**: `RobotCommand.msg`

```python
# RobotCommand.msg
std_msgs/Header header

string command_type          # 指令类型
# "move_to_pose"   - 移动到指定位姿
# "start_task"     - 开始作业
# "stop_task"      - 停止作业
# "set_parameter"  - 设置参数

# 参数（根据command_type使用）
geometry_msgs/Pose target_pose       # 目标位姿
string task_type                      # 任务类型
float32 parameter_value               # 参数值
string parameter_name                 # 参数名称
```

---

#### 1.1.3 紧急停止Topic

**Topic名称**: `/robot/emergency_stop`

**消息类型**: `std_msgs/Bool`

```python
# True = 紧急停止

```

**QoS配置**:
```yaml

```

## 2. Modbus TCP接口

### 2.1 PLC通信协议

#### 2.1.1 连接参数

```yaml
IP地址: 192.168.1.100 (可配置)
端口: 502
从站ID: 1
超时: 1000ms
重连间隔: 5s
```

#### 2.1.2 寄存器映射表

**控制寄存器 (Holding Registers)**

| 地址 | 名称 | 数据类型 | 单位 | 读/写 | 说明 |
|------|------|----------|------|-------|------|
| 0x0000 | 系统控制字 | UINT16 | - | R/W | Bit0:启动 Bit1:停止 Bit2:急停 |
| 0x0001 | 底盘左电机速度 | INT16 | RPM | R/W | -1000 ~ 1000 |
| 0x0002 | 底盘右电机速度 | INT16 | RPM | R/W | -1000 ~ 1000 |
| 0x0003 | 转台角度目标 | INT16 | 0.1° | R/W | -1800 ~ 1800 |
| 0x0004 | 大臂关节1角度 | INT16 | 0.1° | R/W | -900 ~ 900 |
| 0x0005 | 大臂关节2角度 | INT16 | 0.1° | R/W | -900 ~ 900 |
| 0x0006 | 大臂关节3角度 | INT16 | 0.1° | R/W | -900 ~ 900 |
| 0x0010 | 喷涂开关 | UINT16 | - | R/W | 0:关闭 1:开启 |
| 0x0011 | 打磨开关 | UINT16 | - | R/W | 0:关闭 1:开启 |
| 0x0012 | 打磨压力目标 | UINT16 | N | R/W | 0 ~ 500 |

**状态寄存器 (Input Registers)**

| 地址 | 名称 | 数据类型 | 单位 | 读/写 | 说明 |
|------|------|----------|------|-------|------|
| 0x0000 | 系统状态字 | UINT16 | - | R | Bit0:运行中 Bit1:故障 Bit2:急停 |
| 0x0001 | 底盘左电机实际速度 | INT16 | RPM | R | -1000 ~ 1000 |
| 0x0002 | 底盘右电机实际速度 | INT16 | RPM | R | -1000 ~ 1000 |
| 0x0003 | 转台实际角度 | INT16 | 0.1° | R | -1800 ~ 1800 |
| 0x0004 | 大臂关节1实际角度 | INT16 | 0.1° | R | -900 ~ 900 |
| 0x0005 | 大臂关节2实际角度 | INT16 | 0.1° | R | -900 ~ 900 |
| 0x0006 | 大臂关节3实际角度 | INT16 | 0.1° | R | -900 ~ 900 |
| 0x0010 | 故障代码 | UINT16 | - | R | 见故障代码表 |
| 0x0011 | 打磨实际压力 | UINT16 | N | R | 0 ~ 500 |
| 0x0012 | 材料剩余量 | UINT16 | % | R | 0 ~ 100 |

**故障代码表**

| 代码 | 说明 |
|------|------|
| 0x0000 | 无故障 |
| 0x0001 | 通信超时 |
| 0x0002 | 电机过载 |
| 0x0003 | 传感器故障 |
| 0x0004 | 位置超限 |
| 0x0005 | 急停触发 |
| 0x00FF | 未知故障 |

### 2.2 Modbus读写示例

#### 2.2.1 读取底盘速度

```python
# Python示例
from pymodbus.client import ModbusTcpClient

client = ModbusTcpClient('192.168.1.100', port=502)
client.connect()

# 读取输入寄存器 0x0001 和 0x0002（左右电机速度）
result = client.read_input_registers(0x0001, 2, unit=1)

if not result.isError():
    left_speed = result.registers[0]
    right_speed = result.registers[1]
    print(f"左电机: {left_speed} RPM, 右电机: {right_speed} RPM")

client.close()
```

#### 2.2.2 控制底盘移动

```python
# 写入保持寄存器 0x0001 和 0x0002（设置目标速度）
client.write_registers(0x0001, [500, 500], unit=1)  # 前进
```

## 5. 接口调用示例

### 5.1 ROS2 Python示例

#### 5.1.1 发布控制指令

```python
import rclpy
from rclpy.node import Node
from epoxybot_msgs.msg import RobotCommand
from geometry_msgs.msg import Pose

class CommandPublisher(Node):
    def __init__(self):
        super().__init__('command_publisher')
        self.publisher = self.create_publisher(
            RobotCommand, 
            '/robot/cmd', 
            10
        )
    
    def move_to_pose(self, x, y, theta):
        msg = RobotCommand()
        msg.command_type = "move_to_pose"
        msg.target_pose.position.x = x
        msg.target_pose.position.y = y
        msg.target_pose.orientation.z = theta
        
        self.publisher.publish(msg)
        self.get_logger().info(f'移动到 ({x}, {y}, {theta})')

# 使用
rclpy.init()
node = CommandPublisher()
node.move_to_pose(10.0, 5.0, 0.0)
```

#### 5.1.2 订阅机器人状态

```python
class StatusSubscriber(Node):
    def __init__(self):
        super().__init__('status_subscriber')
        self.subscription = self.create_subscription(
            RobotState,
            '/robot/state',
            self.state_callback,
            10
        )
    
    def state_callback(self, msg):
        self.get_logger().info(
            f'机器人状态: {msg.status}, 进度: {msg.progress}%'
        )
```

### 5.2 ROS2 C++示例

#### 5.2.1 调用Service

```cpp
#include <rclcpp/rclcpp.hpp>
#include "epoxybot_msgs/srv/plan_task.hpp"

class TaskPlanner : public rclcpp::Node {
public:
    TaskPlanner() : Node("task_planner") {
        client_ = this->create_client<epoxybot_msgs::srv::PlanTask>(
            "/plan_task"
        );
    }
    
    void plan_spray_task() {
        auto request = std::make_shared<epoxybot_msgs::srv::PlanTask::Request>();
        request->task_type = "spray";
        
        // 等待服务可用
        while (!client_->wait_for_service(std::chrono::seconds(1))) {
            RCLCPP_INFO(this->get_logger(), "等待服务...");
        }
        
        auto future = client_->async_send_request(request);
        
        if (rclcpp::spin_until_future_complete(this->get_node_base_interface(), future)
            == rclcpp::FutureReturnCode::SUCCESS)
        {
            auto result = future.get();
            RCLCPP_INFO(this->get_logger(), "规划成功, 预计耗时: %.2f秒", 
                        result->estimated_time);
        }
    }

private:
    rclcpp::Client<epoxybot_msgs::srv::PlanTask>::SharedPtr client_;
};
```
