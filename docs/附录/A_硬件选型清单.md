## 2. 传感器设备

### 2.1 FAST-LIVO2场景建模套件

**套件描述**：可插拔式传感器套件，安装于侧墙平台大臂末端，用于场景3D建模。

**套件组成**：
- **激光雷达**：Livox Avia
- **相机**：海康 MV-CU013-A0UC（用于视觉融合）
- **IMU惯性测量单元**：集成于STM32F103
- **处理单元**：STM32F103 + Raspberry Pi 4B

| 项目 | 参数 |
|------|------|
| **数量** | 1套（仅侧墙平台使用） |
| **安装方式** | 可插拔于大臂末端 |
| **使用场景** | 侧墙平台走一圈进行全面场景建模 |
| **算法** | FAST-LIVO2（激光+IMU+视觉融合） |
| **参考方案** | https://github.com/xuankuzcr/LIV_handhold |
| **单价** | 待确认（成套采购） |

**选型理由**：
- FAST-LIVO2已经过初步验证，具备可行性
- Livox Avia激光雷达性能优异，适合SLAM应用
- 多传感器融合提高建模精度和鲁棒性
- 可插拔设计便于灵活使用
- 仅需侧墙平台走一圈即可建立全场景模型
- 采用官方推荐硬件组合，相机型号因原型号停产而替换为MV-CU013-A0UC
